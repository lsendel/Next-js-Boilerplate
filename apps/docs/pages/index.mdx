## Quick Start

Get started in 5 minutes:

1. **Clone the repository**
   ```bash
   git clone https://github.com/ixartz/Next-js-Boilerplate.git my-project
   cd my-project
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Run development server**
   ```bash
   pnpm dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```
